# TrueVi — Architecture &amp; Wireframes

Specification repository for the TrueVi page builder system.

**Live site:** https://ffm-truevi.github.io/TrueVi_BP_Arch-Wireframes/

## What this is

A self-contained specification covering the architecture, API contract, data
model, Management Console wireframes, and page template wireframes for the
TrueVi page builder system. All documents are static HTML, viewable directly
in any browser via GitHub Pages.

## Audience

- **TrueVi internal team** — Brian (COO), Bunker (CEO)
- **TrueVi engineering** — Devin and incoming engineers
- **External design and engineering contractors**

This repository is intentionally separated from the TrueVi codebase so
contractors can be granted access to the specification without exposure to
the proprietary application source.

## Milestones

- **M1 · Foundations** — Architecture, API contract, data model
- **M2 · Management Console** — Template builder, selector rules, experiments, settings
- **M3 · Page Templates** — Component library + 6 page template wireframes

## Updating

Specs are living documents. Every revision is committed here. The live site
auto-updates within ~60 seconds of a push to `master`.
